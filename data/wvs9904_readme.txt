wvs9904: World Values Survey (WVS), Wave 4 (1999-2004)
------------------------------------------------------

  Source: v2018-09-12 official release

  http://www.worldvaluessurvey.org/
  http://www.worldvaluessurvey.org/WVSDocumentationWV4.jsp

File manifest
-------------

The wvs9904.zip file contains:

  wvs9904                .dta  -- dataset (Stata 12+ format)
  wvs9904_codebook       .pdf  -- official codebook
  wvs9904_questionnaire  .pdf  -- official questionnaire (English)
  wvs9904_variables      .txt  -- list of all variables

  ... as well as the wvs9904.do Stata do-file, which was used to create the
  wvs9904 dataset, and this README file.

Production context
------------------

The wvs9904.zip file is part of the Statistical Reasoning and Quantitative 
Methods (SRQM) course taught at Sciences Po, Paris:

  https://f.briatte.org/teaching/quanti
  https://github.com/briatte/srqm

IMPORTANT NOTES
---------------

- The wvs9904.dta file is a teaching dataset. It might differ from the 
  official data, and should not be redistributed or used for any other purpose 
  than instruction.

- The codebook provided with the dataset is only a small part of the official 
  documentation that comes with the data: please refer to the official dataset 
  website for more information.

Suggested citation
------------------

The producers of the original datasets recommend citing the data as follows:

> Inglehart, R., C. Haerpfer, A. Moreno, C. Welzel, K. Kizilova, J. 
Diez-Medrano, M. Lagos, P. Norris, E. Ponarin & B. Puranen et al. (eds.). 2014. 
World Values Survey: Round Four - Country-Pooled Datafile Version. Madrid: JD 
Systems Institute.

The links at the top of this README file should also appear in the citation.

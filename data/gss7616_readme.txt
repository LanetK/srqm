gss7616: U.S. General Social Survey (GSS) 1976, 1986, 1996, 2006 and 2016
-------------------------------------------------------------------------

  Source: GSS cross-sectional cumulative data (Release 4, August 16, 2018)

  http://gss.norc.org/
  http://gss.norc.org/Get-Documentation

File manifest
-------------

The gss7616.zip file contains:

  gss7616                .dta  -- dataset (Stata 12+ format)
  gss7616_codebook       .pdf  -- official codebook
  gss7616_variables      .txt  -- list of all variables

  ... as well as the gss7616.do Stata do-file, which was used to create the
  gss7616 dataset, and this README file.

Production context
------------------

The gss7616.zip file is part of the Statistical Reasoning and Quantitative 
Methods (SRQM) course taught at Sciences Po, Paris:

  https://f.briatte.org/teaching/quanti
  https://github.com/briatte/srqm

IMPORTANT NOTES
---------------

- The gss7616.dta file is a teaching dataset. It might differ from the 
  official data, and should not be redistributed or used for any other purpose 
  than instruction.

- The codebook provided with the dataset is only a small part of the official 
  documentation that comes with the data: please refer to the official dataset 
  website for more information.

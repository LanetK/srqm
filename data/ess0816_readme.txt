ess0816: European Social Survey (ESS), Rounds 4 (2008-9) and 8 (2016-7)
-----------------------------------------------------------------------

  Source: ESS4e04_5 (Round 4 edition 4.5) and ESS8e02_1 (Round 8 edition 2.1)

  http://www.europeansocialsurvey.org/
  http://www.europeansocialsurvey.org/data/

File manifest
-------------

The ess0816.zip file contains:

  ess0816                 .dta  -- dataset (Stata 12+ format)
  ess0816_codebook_r*     .pdf  -- official codebooks (Rounds 4 and 8)
  ess0816_weighting_guide .pdf  -- official note on survey weights
  ess0816_variables       .txt  -- list of all variables

  ... as well as the ess0816.do Stata do-file, which was used to create the
  ess0816 dataset, and this README file.

Production context
------------------

The ess0816.zip file is part of the Statistical Reasoning and Quantitative 
Methods (SRQM) course taught at Sciences Po, Paris:

  https://f.briatte.org/teaching/quanti
  https://github.com/briatte/srqm

IMPORTANT NOTES
---------------

- The ess0816.dta file is a teaching dataset. It might differ from the 
  official data, and should not be redistributed or used for any other purpose 
  than instruction.

- The codebook provided with the dataset is only a small part of the official 
  documentation that comes with the data: please refer to the official dataset 
  website for more information.

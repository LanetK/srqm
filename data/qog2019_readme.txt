qog2019: Quality of Government (QOG), January 2019 Standard cross-section
-------------------------------------------------------------------------

  Source: Quality of Government Institute, University of Gothenburg, Sweden

  https://qog.pol.gu.se/
  https://qog.pol.gu.se/data/datadownloads/qogstandarddata

File manifest
-------------

The qog2019.zip file contains:

  qog2019                .dta  -- dataset (Stata 12+ format)
  qog2019_codebook       .pdf  -- official codebook
  qog2019_variables      .txt  -- list of all variables

  ... as well as the qog2019.do Stata do-file, which was used to create the
  qog2019 dataset, and this README file.

Production context
------------------

The qog2019.zip file is part of the Statistical Reasoning and Quantitative 
Methods (SRQM) course taught at Sciences Po, Paris:

  https://f.briatte.org/teaching/quanti
  https://github.com/briatte/srqm

IMPORTANT NOTES
---------------

- The qog2019.dta file is a teaching dataset. It might differ from the 
  official data, and should not be redistributed or used for any other purpose 
  than instruction.

- The codebook provided with the dataset is only a small part of the official 
  documentation that comes with the data: please refer to the official dataset 
  website for more information.

nhis1017: U.S. National Health Interview Survey (NHIS) 2010 and 2017
--------------------------------------------------------------------

  Source: U.S. Centers for Disease Control (CDC), NCHS

  https://www.cdc.gov/nchs/nhis/index.htm
  https://www.cdc.gov/nchs/nhis/data-questionnaires-documentation.htm

File manifest
-------------

The nhis1017.zip file contains:

  nhis1017            .dta  -- dataset in Stata 12+ format
  nhis1017_codebook_* .pdf  -- official codebooks (2010 and 2017)
  nhis1017_variables  .txt  -- list of all variables

  ... as well as the nhis1017.do Stata do-file, which was used to create the
  nhis1017 dataset, and this README file.

Production context
------------------

The nhis1017.zip file is part of the Statistical Reasoning and Quantitative 
Methods (SRQM) course taught at Sciences Po, Paris:

  https://f.briatte.org/teaching/quanti
  https://github.com/briatte/srqm

IMPORTANT NOTES
---------------

- The nhis1017.dta file is a teaching dataset. It might differ from the 
  official data, and should not be redistributed or used for any other purpose 
  than instruction.

- The codebook provided with the dataset is only a small part of the official 
  documentation that comes with the data: please refer to the official dataset 
  website for more information.

Suggested citation
------------------

The producers of the original datasets recommend citing the data as follows:

> National Center for Health Statistics. National Health Interview Survey,
2017. Public-use data file and documentation. 2018.

> National Center for Health Statistics. Data File Documentation, National
Health Interview Survey, 2010 (machine readable data file and documentation).
National Center for Health Statistics, Centers for Disease Control and
Prevention, Hyattsville, Maryland. 2011.

> URL: https://www.cdc.gov/nchs/nhis/data-questionnaires-documentation.htm
